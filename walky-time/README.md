## サイト概要
### サイトテーマ
- 保護犬とのお散歩コミュニティーサイト
​
### テーマを選んだ理由
犬を飼えない環境から、犬と触れ合える時間が欲しく、お散歩のサイトを作ってみようと思いました。
なお、お散歩対象は「保護犬」で、サイトを通じて、お散歩から保護犬を家族に迎え入れることもできるシステムと、
お散歩することで、保護犬も人に慣れる練習ができるため、こちらのテーマといたしました。
​
### ターゲットユーザ
- 犬が大好き！だけど飼えない方
- 犬を迎え入れたい方
​
### 主な利用シーン
- 犬と触れ合いたい時
- 迎え入れたい！けど、触れ合って決めたい
